# AZ-305 Azure Solutions Architect Expert — Skill Deep Dive (Synthetic)

> Synthetic study reference for demonstration only. (Prerequisite: AZ-204.)

## Architecture Design

Designing solutions on Azure against the Well-Architected Framework (reliability,
security, cost, operational excellence, performance efficiency). Topics:
choosing compute (VMs, App Service, AKS, Functions), high availability and
disaster recovery, availability zones/regions, and trade-off analysis.

## Cost Optimization

Designing for cost. Topics: right-sizing, reservations and savings plans, spot
capacity, autoscaling, storage tiering, Azure Cost Management budgets and alerts,
and choosing managed services to reduce operational overhead.

## Identity

Identity and access design. Topics: Microsoft Entra ID, conditional access,
managed identities, RBAC vs ABAC, B2B/B2C, privileged identity management (PIM),
and designing zero-trust access for workloads.

## Data Solutions

Designing data storage and integration. Topics: choosing between SQL Database,
Cosmos DB, Storage, and Synapse; consistency models; partitioning; data
residency and encryption; and designing analytics/data-platform architectures.

## Networking

Designing network topology and connectivity. Topics: hub-and-spoke, VNet
peering, private endpoints, Azure Firewall, Application Gateway/Front Door,
ExpressRoute vs VPN, and securing traffic with NSGs and segmentation.
